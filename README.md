# Discord Bot Legal Documents

This repository contains the Terms of Service and Privacy Policy for our Discord bot.

## Files

- **index.html** - Landing page with links to legal documents
- **terms.html** - Terms of Service
- **privacy.html** - Privacy Policy

## GitHub Pages

This repository is configured to be served via GitHub Pages. To enable:

1. Go to repository Settings
2. Navigate to Pages
3. Select source branch (usually `main` or `master`)
4. Save

The site will be available at: `https://[username].github.io/discord-bot-legal/`
